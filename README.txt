Данный проект является простым наглядным примером моих навыков в Java с использованием Spring Boot.
В проекте вы можете увидеть:

RESTful взаимодействие с проектом.
Работа с базой данных, по средством настройки сущностей и выставление анотаций.
Тестирование отдельных классов.
Интеграционное тестирование приложения.
Http запросы к дургим сервисам.
Приложение демонстрирует общение двух микросервисов между собой.
